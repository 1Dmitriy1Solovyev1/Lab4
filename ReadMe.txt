Задание: Написать программу по созданию, добавлению, просмотру и решению поставленной задачи
для однонаправленного линейного списка типа СТЕК.Реализовать сортировку стека двумя рассмотренными выше методами.
Во всех заданиях создать список из положительных и отрицательных случайных целых чисел.
В созданном списке поменять местами крайние элементы.
1. Однонаправленная очередь - очередь на базе линейного однонаправленного списка, линейный список, в котором включения элементов производятся на одном конце списка, а все исключения делаются на другом его конце.
2. Дек (deque — double ended queue, «двусторонняя очередь») – структура данных типа «список», функционирующая одновременно по двум принцам организации данных: FIFO и LIFO. 
3. Удобно использовать при создании больших по памяти объектов, для избежания излишнего выделения памяти.
